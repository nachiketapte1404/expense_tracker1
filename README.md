Expense Tracker

A full stack project for expense management

Instructions for running:

1. run npm install in both frontend and backend folders individually <br>
2. create a .env file in backend folder <br>
3. add the following credentials to it:<br>
    PORT=5000 <br>
    CORS_ORIGIN=http://localhost:5173<br>
    JWT_SECRET=<jwt secret token><br>
    DB_HOST=127.0.0.1<br>
    DB_USER=username<br>
    DB_PASSWORD=password<br>
    DB_NAME=db_name<br>
